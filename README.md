# crow use template
是一个自用的crow使用模板<br>

## 用了哪些库?
> <pre><code>- boost
- openssl
- sqlite3
- sqlite-modern-cpp
- cryptopp
- asyncio
- nlohmann-json
- yaml-cpp
- spdlog
- fmt
- crow
- poco
- bcrypt-cpp
- generator_coroutine
- snowflake-cpp
</code></pre>

※使用了vcpkg需要自己手动配置cmake的vcpkg目录


## C++标准是多少?
C++20

## 怎么编译

<pre><code>$ mkdir build && cd build
$ cmake ..
$ make
</code></pre>

项目最终会生成到<b>bin</b>目录
